This site is a template project for bootstrapping start-up sites that require user accounts.

The frameworks in use are Django and Material UI CSS.

I'm not a big fan of modern single-page applications, frameworks like React, and Angular, I think they're way too complicated for small startups.

So I created this boilerplate. The code can be found at https://github.com/jackmurray90/django-mui.
